## üìå Instructions:

Instructions tailored for creating an HTML static page:

### 1. Profile Content:

1. **Create Profile HTML File:**
   - Open a text editor or code editor on your computer.
   - Create a new file and name it `YourName_Profile.html`.

2. **Include Essential Elements:**
   - Begin by writing the HTML boilerplate code.
   - Inside the `<body>` tag, structure your profile page with appropriate HTML elements.
   - Include a `<header>` section with your name and current position/title.
   - Write a brief introduction about yourself within a `<section>` or `<div>` element.
   - List the projects you have worked on using unordered or ordered lists (`<ul>` or `<ol>`).
   - Highlight your achievements with descriptive `<p>` elements or headings.
   - Add your contact information, such as email address and LinkedIn profile, within a separate `<footer>` section.

3. **Personalize Your Profile Content:**
   - Customize the content of each section to reflect your unique experiences and skills.
   - Use semantic HTML elements to enhance accessibility and search engine optimization (SEO).
   - Incorporate CSS styles or inline styles to enhance the visual appeal of your profile page.
   - Include images or icons to make your profile visually engaging and memorable.

### 2. Location:

1. **Store Profile File:**
   - Navigate to the Pero's Academy Assignments repository on GitHub.
   - Locate the folder named `your_github.io_professional_site_assignment`.
   - Upload your `YourName_Profile.html` file to this folder.

2. **Accessibility:**
   - Ensure that your HTML file is easily accessible for review by instructors and peers.
   - Verify that the file permissions allow others to view and comment on your work.
   - Consider adding a link to your profile page in the README.md of the assignments repository for easy navigation.

### Submission:

1. **Commit Profile File:**

   - **Save Changes:** After completing the writing and styling of your profile page, ensure that all changes are saved to your HTML file.
   
   - **Stage Changes:** Open your terminal or Git client and navigate to the directory containing your Pero's Academy Assignments repository.
   
   - **Stage Files:** Use the `git add` command to stage your profile file for commit. For example:
     ```
     git add YourName_Profile.html
     ```
   
   - **Commit Changes:** Once the file is staged, commit your changes with a descriptive message using the `git commit` command. For example:
     ```
     git commit -m "Add profile page for YourName"
     ```

   - **Push Changes:** Push your committed changes to the remote repository using the `git push` command. For example:
     ```
     git push origin main
     ```

2. **Notify Instructor:**

   - **Send Notification:** After committing your profile file, notify your instructor of your submission promptly.
   
   - **Communication Method:** You can notify your instructor via email, direct message on the learning platform, or any other communication channel specified by your instructor.
   
   - **Include Information:** In your notification, mention that you have completed the assignment and provide a direct link to your profile page in the Pero's Academy Assignments repository.
   
   - **Additional Instructions:** If your instructor has provided any specific instructions or requirements for submission notification, ensure that you follow them accordingly.

3. **Review Deadline:**

   - **Deadline Awareness:** Be aware of the deadline specified by your instructor for assignment submission.
   
   - **Submission Timing:** Aim to submit your profile page well before the deadline to allow sufficient time for review and feedback.
   
   - **Avoid Last-Minute Submissions:** Avoid waiting until the last minute to submit your assignment to prevent potential technical issues or delays.

By following these detailed submission instructions, you can ensure that your profile page is successfully submitted, reviewed, and evaluated by your instructor within the specified deadline.
